# Nest Template

1. Instalar dependencias
```npm i```
2. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
3. Cambiar las variables de entorno
4. Levantar la base de datos
```
docker-compose up -d
```
5. Levantar: ```npm run start:dev```
